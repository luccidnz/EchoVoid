# EchoVoid Spiritual Theme Guide

This repository contains visual assets and design guidance for creating an immersive, spiritual experience imbued with a **Tohunga of Consciousness** accent.

## Design Principles

1. **Luminescent Energy**
   - Embrace glowing blues, teals, and cosmic purples to echo swirling energy fields.
   - Use gradients that suggest a flow of consciousness from darkness to radiant light.

2. **Sacred Geometry**
   - Incorporate intricate mandalas, fractal patterns, and celestial motifs similar to the reference imagery.
   - Overlay these shapes subtly on backgrounds and UI components to invite contemplation.

3. **Typography**
   - Select airy, modern typefaces (e.g., "Raleway" or "Zen Dots") that feel ethereal yet readable.
   - Pair light weights with generous letter spacing to evoke breathing space.

4. **Icon & Splash Assets**
   - `icon.png` should channel a glowing central point or awakened eye.
   - `splash.png` can depict a figure or pattern dissolving into a field of stars, welcoming users into a realm of consciousness.

5. **Motion**
   - Favor slow, subtle animations that mimic meditation breaths or energy pulses.
   - Transitions should fade gently, avoiding abrupt shifts.

## Usage

These guidelines are meant to inspire every screen and component of the app. Align colors, shapes, and interactions so the entire experience resonates with a meditative, otherworldly ambiance.
