# VOLT-MAN (Web Playable Build)

A browser-playable implementation of the VOLT-MAN maze-action game concept.

## Run locally

```bash
python -m http.server 4173
```

Open: `http://localhost:4173`

## Controls

- Move: Arrow keys / WASD
- Trigger overcharge: Space (requires 100% charge)

## Included systems

- Story comic intro sequence
- Story / Quick / Endless / Challenge mode launch paths
- Multi-world progression from JSON configs
- 4 enemy tiers with distinct chase styles
- Hazards: slime, traps, portals, carrot missiles
- Power-ups: Volt-Carrot, Bone, Glove, Stopwatch
- Bonus items: relics, collars, masks
- HUD + charge meter + score/lives/combo tracking
